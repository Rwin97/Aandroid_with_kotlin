Resumen
Para crear un proyecto nuevo, inicia Android Studio, haz clic en + Start a new Android Studio project, asigna un nombre a tu proyecto, selecciona una plantilla y completa los detalles.
Para crear un dispositivo virtual de Android (un emulador) que ejecute tu app, elige Tools > Device Manager y usa el Administrador de dispositivos para seleccionar un dispositivo de hardware y una imagen del sistema.
Para ejecutar tu app en un dispositivo virtual, asegúrate de haber creado uno, selecciónalo en el menú desplegable de la barra de herramientas y, luego, haz clic en el ícono Ejecutar  Run app o haz clic en el ícono de Run [ICON HERE] ubicado en la barra de herramientas.
